# CapstoneProject

📌 Event Registration System

A full-stack web application that allows users to register for events seamlessly. This system handles user data collection, storage, and retrieval using a structured backend and database integration.

🚀 Features
📝 User Registration Form
📦 Backend API with Express.js
🗄️ Database Integration for storing user data
🔄 RESTful API communication
✅ Form validation and structured data handling
📊 Organized data storage (Name, Age, Course, Comments, etc.)

⚙️ How It Works:
User fills out the registration form on the frontend
Form data is sent to backend using HTTP requests
Express.js API receives and processes the data
Data is stored in the database
Response is sent back to frontend
🔌 API Integration:
RESTful APIs are used for communication between frontend and backend
Example endpoint:
POST /register
Handles:
Receiving user data
Validating inputs
Storing in database
🗃️ Database Schema
Field	Type
Name	VARCHAR
Age	INT
Course	VARCHAR
Comments	TEXT


▶️ Installation & Setup
1. Clone the repository
git clone https://github.com/JayVinodThakor/CapstoneProject
2. Navigate to project folder
cd event-registration-system
3. Install dependencies
npm install
4. Start the server
node server.js
5. Open frontend

Just open index.html in your browser
